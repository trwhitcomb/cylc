A.1 Top Level Items

The only top level configuration items at present are the suite title and description.

A.1.1 title

The suite title is displayed in the gcylc suite database window. It can also be retrieved from a suite at run time with cylc show (or use cylc get-config).

  • type: string
  • default: “No title provided”

A.1.2 description

The suite description can be retrieved by gcylc right-click menu. It can also be retrieved from a suite at run time with cylc show (or use cylc get-config).

  • type: string
  • default: “No description provided”
